# 2. ground_truth labeling 생성
- 다음 단계: Ground Truth 구축

1단계: Cohen's Kappa 계산
평가자 2명의 일치도 측정

2단계: 불일치 조정
Kappa < 0.6이면 불일치 항목 재검토 및 합의
3단계: 최종 Ground Truth 확정
합의된 라벨을 정답 데이터로 저장

4단계: NCS 참조 문서 준비

7개 세분류별 참조 텍스트 생성 (세분류명 + 직무정의 + 능력단위)

5단계: 자동 매핑 실험

Ko-SBERT 실험 (1:1, 1:3)
GPT-4o-mini 실험 (1:1, 1:3)

6단계: 성능 평가
Accuracy, Precision, Recall, F1, Confusion Matrix
